Guida all'utilizzo

1. Home Screen (Schermata Principale)

Questa è la dashboard centrale dell'applicazione.

  • Pulsante Condividi (Icona in alto a destra): Permette di condividere l'URL dell'applicazione tramite le opzioni di sistema o WhatsApp.
  • REPORT PER LA FILIALE: Genera un riepilogo statistico (totale attivi, pionieri, studi, ecc.) destinato alla filiale.

Barra Strumenti Rapida:

  • SYNC: Avvia la sincronizzazione automatica con il cloud (se configurata).
  • DOPPI: Funzionalità per identificare e unire moduli duplicati.
  • LINK: Genera o condivide un link diretto al modulo.

Pulsanti Principali:

  • Nuovo Modulo S-21: Apre un modulo vuoto per inserire un nuovo proclamatore.
  • Anagrafica Totale: Visualizza l'elenco completo di tutti i proclamatori inseriti.
  • Gruppi di Servizio: Accede alla gestione dei gruppi (assegnazione sorveglianti e membri).
  • Storico Annuale: Permette di consultare i dati degli anni di servizio precedenti archiviati.
  • Backup JSON: Esporta l'intero database locale in un file .json per sicurezza.
  • Importa Backup: Carica un file .json precedentemente salvato per ripristinare i dati.

2. Gestione Elenco (List Screen)

Accessibile tramite "Anagrafica Totale" o i filtri rapidi.

  • Barra di Ricerca: Filtra i nomi dei proclamatori in tempo reale.
  • Filtri Avanzati (Toggle): Espande una sezione per ricerche mirate.
  • Checkbox INCARICHI: Filtra per Anziano, Servitore, P. Regolare o Speciale.
  • Checkbox CARATTERISTICHE: Filtra per sesso (Maschio/Femmina) e speranza (Altre Pecore/Unto).
  • Menu a tendina Rapporti: Permette di isolare i "Rapporti Mancanti", "Irregolari" o "Inattivi".
  • Resetta Filtri: Ripristina la visualizzazione completa.

Azioni di Gruppo (Header):

  • ZIP PDF: Genera un archivio compresso contenente i PDF di tutti i proclamatori filtrati.
  • Stampa: Apre la finestra di stampa del browser per la lista corrente.

3. Modulo S-21 (Scheda Proclamatore)

L'interfaccia di inserimento dati che ricalca il modulo cartaceo ufficiale.

Intestazione e Dati Anagrafici:

  • Nome e cognome / Data di nascita / Data del battesimo: Campi di testo per l'identificazione.
  • Checkbox Tipologia: Selezione di Sesso (M/F) e Speranza (Altre Pecore/Unto).
  • Checkbox Incarichi: Selezione del ruolo teocratico (Anziano, Servitore, vari tipi di Pioniere).

Tabella Ministero (Dati mensili):

  • ANNO DI SERVIZIO: Campo modificabile per indicare l'anno corrente (es. 2025).
  • Partecipazione: Checkbox per indicare se il proclamatore ha fatto rapporto nel mese.
  • Studi Biblici: Numero di studi tenuti nel mese.
  • Pioniere Ausiliario: Spunta se il proclamatore ha servito come ausiliario in quel mese.
  • Ore: Inserimento delle ore (obbligatorio per pionieri/missionari). Il Totale Ore in fondo alla tabella viene calcolato automaticamente.
  • Osservazioni: Note libere per ogni singolo mese.

Controlli Modulo:

  • Salva: Memorizza le modifiche nel database locale.
  • PDF: Genera e scarica immediatamente la scheda S-21 in formato PDF professionale.
  • Zoom Slider: Permette di ingrandire o rimpicciolire l'anteprima del modulo su dispositivi mobili.
  • Archivia e Nuovo Anno: Sposta i dati correnti nello storico e pulisce la tabella per il nuovo anno di servizio.

4. Gestione Gruppi (Groups Screen)

  • Nuovo Gruppo: Crea un nuovo raggruppamento.
  • Titolo Gruppo: Nome identificativo (es. "Gruppo 1").
  • Sorvegliante / Assistente: Menu a tendina per selezionare i fratelli responsabili dall'elenco anagrafico.
  • Componenti Disponibili: Elenco con checkbox per aggiungere o rimuovere membri dal gruppo.
